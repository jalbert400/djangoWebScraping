# PROYECTO DEEPLINK EN DJANGO

## REQUERIMIENTOS


```
La aplicación web scrapping con selenium y tkinter de python nos armará la estructura de un deeplink

Condiciones:
El usuario debe tener instalado el navegador chrome
La primera vez debe logearse con sus accesos para tener registrado sus accesos

Tiempo de Respuesta:
Se hizo pruebas únicas y multiples siendo el promedio el tiempo de 35 - 40 segundos
```
